# Task Management System

A full-stack Task Management System built using MERN stack.

---

## Features
- User Registration & Login (JWT Authentication)
- Create, Update, Delete Tasks
- Filter tasks by status
- Protected Routes

---

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

---

## ⚙️ Setup Instructions

### Backend
cd backend  
npm install  
npm start  

### Frontend
cd frontend  
npm install  
npm run dev  

---

## API Routes

### Auth
POST /api/register  
POST /api/login  

### Tasks
GET /api/tasks  
POST /api/tasks  
PUT /api/tasks/:id  
DELETE /api/tasks/:id
